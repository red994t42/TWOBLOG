Cara penggunaan GNU Octave mirip dengan MATLAB karena Octave didesain untuk kompatibilitas dengan MATLAB. Berikut adalah langkah-langkah dasar untuk memulai dengan Octave:

1. Instalasi Octave:

  • Windows: Unduh installer dari situs web resmi Octave dan ikuti petunjuk instalasinya.
  • MacOS: Octave tersedia melalui Homebrew atau MacPorts, atau Anda bisa mengunduh paket installer dari situs web resmi Octave.
  • Linux: Banyak distribusi Linux menyertakan Octave dalam repositori mereka. Anda bisa menginstal melalui manajer paket favorit Anda (misalnya, apt-get untuk Ubuntu).

2. Mulai Octave:

  • Windows: Buka Octave dari shortcut yang dibuat saat instalasi atau dari menu Start.
  • MacOS/Linux: Buka terminal dan ketik octave lalu tekan Enter.

3. Antarmuka Baris Perintah:

  • Saat Octave terbuka, Anda akan melihat antarmuka baris perintah (octave:1>). Di sini Anda dapat memasukkan perintah Octave secara langsung.

4. Menggunakan Perintah dan Fungsi:

  • Anda dapat menggunakan berbagai perintah dan fungsi matematika. Contoh: 1+1, sqrt(9), sin(pi/2), A = [1, 2; 3, 4], dll.

5. Menulis dan Menjalankan Skrip:

  • Buat file teks dengan ekstensi .m. Misalnya, myscript.m.
  • Tulis kode Octave Anda di dalamnya. Contoh:
    octave
    A = [1, 2; 3, 4]; b = [5; 6]; x = A \ b; disp(x);
  • Simpan file dan jalankan di antarmuka baris perintah Octave dengan mengetik nama file tanpa ekstensi (misalnya, myscript).

6. Membuat Fungsi:

  • Anda dapat membuat fungsi dalam file terpisah atau langsung di antarmuka baris perintah.
  • Contoh membuat fungsi:
    octave
    function y = myfunction(x) y = x^2 + 2*x + 1; endfunction
  • Panggil fungsi tersebut dengan menyebut namanya di antarmuka baris perintah Octave.

7. Membuat Plot:

  • Octave memiliki fungsi plot yang mirip dengan MATLAB.
  • Contoh:
    octave
    x = linspace(0, 2*pi, 100); y = sin(x); plot(x, y);

Itulah langkah-langkah dasar untuk memulai menggunakan GNU Octave
